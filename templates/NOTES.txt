
Merci d'avoir installé {{ .Chart.Name }}!

Namespace : {{ .Release.Namespace }}
Domaine   : {{ .Values.exposed_external_url }}

Vérifications :
   1) Pods
     kubectl -n {{ .Release.Namespace }} get pods -o wide

   2) Services
     kubectl -n {{ .Release.Namespace }} get svc

   3) Ingress (Traefik)
     kubectl get ingress
     kubectl describe ingress <nom ingress>

   4) Accès HTTP(S)
     http(s)://examlinux.jaffleman.tech/docs

   5) Vérifions à présent le certificat au bout de quelques secondes 
      kubectl get certificate

Astuce :
  - Logs FastAPI :
    kubectl -n {{ .Release.Namespace }} logs deploy/fastapi -f

  - Postgres :
    kubectl -n {{ .Release.Namespace }} exec -it sts/postgres -- sh -lc 'psql -U "$POSTGRES_USER" -d "$POSTGRES_DB" -c "\l"'
